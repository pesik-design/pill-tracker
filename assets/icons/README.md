# Іконки типів ліків

Дуотон-іконки за **формою** препарату. Малюються через `currentColor`, тому
адаптуються під будь-який колір тайла (заливка — той самий колір з opacity).

| Файл | Форма |
|------|-------|
| `tablet.svg` | Таблетка |
| `capsule.svg` | Капсула |
| `drops.svg` | Краплі |
| `suppository.svg` | Свічка |
| `syrup.svg` | Сироп |
| `injection.svg` | Ін'єкція |
| `spray.svg` | Спрей |

## Логіка призначення
Іконка обирається **автоматично за полем «Форма»** (не за назвою ліків).
Розрізнення ліків однакової форми — через **колір мітки** (обирає користувач).
За наявності **фото упаковки** — показуємо фото замість іконки.

## Використання
```html
<span style="color:#3a86e0">
  <!-- inline вміст capsule.svg -->
</span>
```
viewBox `0 0 24 24`. Задавай розмір через width/height, колір — через `color`.
