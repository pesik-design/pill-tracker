# 💊 пілюлі — трекер прийому ліків

Універсальний застосунок для відстеження прийому ліків та вітамінів.
Локальний, приватний, працює офлайн, встановлюється як PWA на ПК і телефон.

### 🔗 Відкрити застосунок: **https://pesik-design.github.io/pill-tracker/**

> **Статус:** робочий MVP, опубліковано на GitHub Pages.

## 🔒 Приватність
Застосунок **не збирає й не надсилає жодних даних**. Усе, що ти відмічаєш,
зберігається лише у твоєму браузері (`localStorage`) на твоєму пристрої.
Немає акаунтів, серверів і аналітики.

## Що це
- Заводиш будь-які ліки з будь-яким розкладом (щодня, через день, дні тижня, цикли).
- Прийоми прив'язуються до **подій дня** (пробудження, їжа, сон), а не до жорсткого часу — застосунок сам рахує реальний час із твого режиму.
- Відмічаєш прийом: **Прийнято**, **Пізніше** (snooze) або авто-**Пропущено**.
- Курси з терміном, паузами та підрахунком «за дозами».
- Статистика дотримання, ручні важливі нагадування з фото.
- Світла/темна тема, десктоп і мобільна розкладки.

## Дані
Усе зберігається локально в браузері (`localStorage`). Без акаунтів і серверів.
Перенесення між пристроями — через експорт/імпорт JSON.

## Специфікація
Повний план функціоналу, модель даних та архітектура зберігаються локально
(`docs/specification.html`) і не публікуються.

## Структура
```
pill-tracker/
├─ index.html
├─ manifest.webmanifest      · PWA-маніфест
├─ sw.js                     · service worker (офлайн)
├─ styles/  (tokens.css, app.css)
├─ js/
│  ├─ main.js  store.js  schedule.js  onboarding.js
│  ├─ views/   (today, addMed, stats, reminders, settings)
│  └─ components/
└─ icons/                    · 192/512, maskable, apple-touch, favicon
```

## Запуск (для розробки)
Через локальний сервер (PWA потребує http, не `file://`):
```bash
python -m http.server 8000
# відкрити http://localhost:8000
```
